Requisitos previos
- Un dispositivo con sistema operativo Android.
- El archivo APK que se encuentra en este mismo repositorio.
- Espacio de almacenamiento suficiente en el dispositivo.
- Permisos para instalar aplicaciones desde fuentes externas (si corresponde).

  Instalar el APK desde el dispositivo Android

Paso 1: Transferir o descargar el APK

Paso 2: Habilitar la instalación desde fuentes desconocidas

Dependiendo de la versión de Android:

Android 8.0 (Oreo) o superior
- Abre el archivo APK.
- Android mostrará una advertencia indicando que la aplicación no tiene permiso para instalar aplicaciones desconocidas.
- Pulsa Configuración.
- Activa la opción Permitir desde esta fuente.
- Regresa al instalador.
  
Android 7.0 (Nougat) o inferior
- Ve a Ajustes → Seguridad.
- Activa Fuentes desconocidas.
- Confirma la advertencia.

Paso 3: Instalar la aplicación
- Localiza el archivo APK.
- Tócalo para iniciar la instalación.
- Pulsa Instalar.
- Espera a que finalice el proceso.
- Selecciona Abrir o Listo.
